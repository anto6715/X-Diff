Add `xdiff plot REF CMP -v VAR -o out.png` to render the difference between two netCDF fields to a static PNG/PDF/SVG — one full-size map per variable on a cartopy map (coastlines, filled land, lon/lat gridlines, cropped to the data domain), diverging colormap centered at 0 with a robust color limit. Falls back to a plain lon/lat map when the coastline dataset is unavailable.
