Add the live interactive mode to `xdiff plot`: run it without `-o` to serve the plots on a local Panel/Bokeh server (`--port`, `--no-open`). Each variable's difference is drawn on a cartopy/geoviews map — coastlines, filled land, cropped to the data domain — datashaded so it re-aggregates as you zoom (scroll to zoom) and scales to large grids; a slider adjusts the colour limit live (zoom preserved), and hover reads off values. The reference and comparison maps sit in a collapsed card at the bottom. When the coastline dataset can't be reached, the maps fall back to a plain lon/lat view.
